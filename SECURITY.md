# Security Policy

We take the security of Vermithor seriously. If you discover a security vulnerability within this project, please report it to us as soon as possible.

## Reporting a Vulnerability

To report a security vulnerability, please send an email to [security@example.com](mailto:security@example.com) (replace with actual security contact).

Please include the following information:

*   A clear and concise description of the vulnerability.
*   Steps to reproduce the vulnerability.
*   The potential impact of the vulnerability.
*   Any proof-of-concept code or exploits.

We will acknowledge your report within 48 hours and provide a more detailed response within 5 business days.

## Disclosure Policy

We request that you do not disclose any vulnerabilities publicly until we have had a chance to address them. We will work with you to coordinate a public disclosure once the issue has been resolved.

## Our Commitment

We are committed to:

*   Promptly addressing all reported vulnerabilities.
*   Keeping you informed of our progress.
*   Crediting researchers for their responsible disclosure.

Thank you for helping to keep Vermithor secure!
